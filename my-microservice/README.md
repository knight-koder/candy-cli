## ⚡ Quick Start

### 1. Infrastructure Setup
Start the required databases, message brokers, and sidecars:
```bash
docker-compose up -d
```

### 2. Environment Configuration
Initialize your environment variables:
```bash
cp .env.example .env
```

### 3. Start Development Server
```bash
npm install
npm run start:dev
```
